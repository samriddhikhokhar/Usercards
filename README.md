# 📇 Usercards

This is a simple **User Card Design** project created using **HTML and CSS**.

It displays profile cards with:

* Profile image
* User name
* Short description
* Follow button

## 🌐 Live Demo

View the project here:
https://samriddhikhokhar.github.io/Usercards/

## 🛠️ Technologies Used

* HTML
* CSS

## 📁 Project Files

* `index.html` – Main HTML file
* `style.css` – CSS styling file

## 🎯 Purpose of the Project

This project was created to practice:

* HTML structure
* CSS styling
* Card layout design
* Basic UI design concepts

## 📌 How to Use

1. Download or clone the project
2. Open `index.html` in your browser
3. Edit and customize as needed

---

✨ This project is beginner-friendly and great for practicing front-end design basics.
